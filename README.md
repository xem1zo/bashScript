## Задания по Bash-программированию

В этом репозитории находятся решения учебных заданий по Bash-скриптам.
Все скрипты создавались и редактировались в VSCode, репозиторий предназначен для публикации на GitHub или Gitflic.

### Список скриптов

- **`greet.sh`**  
  Запрашивает имя пользователя и выводит приветствие.  
  **Запуск:**  
  ```bash
  bash greet.sh
  ```

- **`sum_calculator.sh`**  
  Запрашивает два числа и выводит их сумму (поддерживаются целые и вещественные числа).  
  **Запуск:**  
  ```bash
  bash sum_calculator.sh
  ```

- **`even_odd.sh`**  
  Определяет, является ли введённое число чётным или нечётным.  
  **Запуск:**  
  ```bash
  bash even_odd.sh
  ```

- **`create_web_project.sh`**  
  Создаёт структуру папок для веб‑проекта вида:
  ```text
  my-project/
  ├── index.html
  ├── css/
  │   └── style.css
  └── js/
      └── script.js
  ```
  Имя корневой папки проекта запрашивается у пользователя.  
  **Запуск:**  
  ```bash
  bash create_web_project.sh
  ```

- **`line_counter.sh`**  
  Подсчитывает количество строк в указанном файле. Путь к файлу передаётся как аргумент.  
  **Пример запуска:**  
  ```bash
  bash line_counter.sh somefile.txt
  ```

- **`password_generator.sh`**  
  Генерирует случайный пароль длиной 8 символов (латинские буквы и цифры).  
  **Запуск:**  
  ```bash
  bash password_generator.sh
  ```

- **`find_by_extension.sh`**  
  Ищет файлы по указанному расширению в текущей директории и всех поддиректориях.  
  **Пример запуска:**  
  ```bash
  bash find_by_extension.sh sh
  ```

### Как запустить скрипты

1. Убедитесь, что у вас установлен Bash (Linux, macOS или Git Bash/WSL в Windows).  
2. При необходимости выдайте права на выполнение:
   ```bash
   chmod +x *.sh
   ```
3. Запускайте нужный скрипт:
   ```bash
   ./greet.sh
   ./sum_calculator.sh
   # и т.д.
   ```

### Используемые технологии

- **Bash** — для написания скриптов;
- **Git** и **GitHub/Gitflic** — для версионного контроля и хранения кода;
